# API Liga con React

## API Rest de equipos y jugadores de la liga, permite realizar las operaciones CRUD en ambas entidades

## Requisitos previos

- Node.js (v.16 o superior)
- Backend API corriendo en localhost:8080

[(Link de la API)](https://github.com/JaviMP-03/API_Liga_Spring_Boot)

## Tecnologías

- React 18
- TypeScript
- Vite
- React Router
- Bootstrap 5
